# Tranqil

## Relax to the soothing sounds of nature 

![tranquil](http://i.imgur.com/kCg0HTR.png)

Click on the images to play the sound, scroll on them to adjust the volume

### Install 
#### Dependencies  

valac 
cmake 
libgtk-3-dev 
libgranite-dev
libnotify-dev
libgstreamer1.0-dev

#### Build 

cd agenda-tasks
mkdir build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=/usr
make
sudo make install
